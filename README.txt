NeverLostDog en una aplicación web desplegada en Heroku y desarrollada mediante NodeJs y 
MongoDB  para generar codigos QR personalizados para tus perros.


-----Aprovisionamiento(Requisitos)------
Tener instalado mongoDB
Tener instalado Node
Tener instalado gestor de paquetes de Node(npm)


-----Ejecución de la app-----
1.Iniciar mongodb
2.npm install (para las dependencias)
3.node src/index.js o heroku local web (Arrancar servidor)
4.Dirigirse a http://localhost:3000/ o a http://localhost:5000/ en caso de utilizar heroku.
